Convert sources to Org-mode
===

```
usage: src2org [-h] [-o OUTPUT] [-t TITLE] path

Convert source code to org-file

positional arguments:
  path                  Path to sourse code

options:
  -h, --help            show this help message and exit
  -o OUTPUT, --output OUTPUT
                        Output file name, default a.org
  -t TITLE, --title TITLE
                        Org file title
```
